# 智填志愿 — 高考志愿填报系统

> 参考 [Pilot1799/Zhangxuefeng-AI-gaokao](https://github.com/Pilot1799/Zhangxuefeng-AI-gaokao) 项目构建的完整 Web 版志愿填报系统

---

## 项目简介

这是一个基于 HTML/CSS/JavaScript 开发的**纯前端高考志愿填报系统**，无需任何后端服务，直接在浏览器中打开即可使用。

---

## 功能模块

| 页面 | 文件 | 功能 |
|------|------|------|
| 首页 | `index.html` | 系统门户、院校数据表格、用户评价 |
| 成绩分析 | `score.html` | 输入分数→计算排名→冲稳保划分 |
| 院校查询 | `school.html` | 多条件筛选院校（含40所院校数据） |
| 智能推荐 | `recommend.html` | 三步骤表单→生成个性化志愿方案 |
| AI咨询 | `ai-chat.html` | 模拟张雪峰风格的智能问答系统 |
| 填报指南 | `guide.html` | 平行志愿规则、冲稳保策略、FAQ |
| 院校对比 | `compare.html` | 多院校横向数据对比 + 图表 |
| 登录 | `login.html` | 用户登录界面 |
| 注册 | `register.html` | 用户注册界面 |

## 快速开始

1. 直接用浏览器打开 `index.html` 即可
2. 所有功能均可在本地运行，无需联网（除非接入真实AI API）
3. 推荐使用 Chrome / Edge 浏览器以获得最佳效果

---

## 技术特点

- **纯前端**：HTML5 + CSS3 + 原生 JavaScript，零依赖，开箱即用
- **响应式设计**：适配桌面、平板、手机等多种屏幕尺寸
- **模拟AI咨询**：内置关键词识别逻辑，模拟张雪峰风格的直接、实用建议
- **数据可视化**：进度条图表展示成绩分段、录取分数对比
- **本地存储**：使用 localStorage 保存用户输入，刷新不丢失

---

## 数据说明

- 内置 **40 所代表性院校**数据（985/211/双非，覆盖广东及主要省份）
- 分数线数据基于 **2025 年广东省** 参考数据，仅供参考
- 排名估算使用**正态分布近似算法**，实际排名以省招生考试院为准

## 扩展建议

若要升级为真实系统，可扩展以下功能：

1. **后端数据库**：将院校数据存入 MySQL，通过 API 查询
2. **真实AI接入**：对接小米MiMo、文心一言等大模型API实现真实对话
3. **用户系统**：Node.js/Python 后端实现真实注册登录
4. **数据更新**：爬取省招考院官网，每年更新录取分数线
5. **导出功能**：使用 jsPDF 生成 PDF 格式的志愿方案
